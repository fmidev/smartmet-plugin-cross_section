# smartmet-plugin-cross_section

Part of [SmartMet Server](https://github.com/fmidev/smartmet-server). See the [SmartMet Server documentation](https://github.com/fmidev/smartmet-server) for a full overview of the ecosystem.

## Overview

The cross-section plugin generates vertical cross-section weather products from gridded data. Given a geographic path and a set of parameters, it produces images showing the vertical profile of atmospheric conditions along the path.

## Features

- Vertical cross-section image generation from QueryData
- Configurable meteorological parameter selection
- Support for pressure and height level cross-sections
